 DESCRIPCION

	Este proyecto, creado a partir de un framework llamado django, 
	es una página web que representa una academia de boxeo.
	La misma cumple diversas funcionalidades como:
	permitir que el usuario se registre como boxeador,
	entrenador o árbitro; mostrar los registros de cada tipo
	(boxeador, entrenador o arbitro); y una funcion de busqueda,
	la cual filtra resultados en base a lo ingresado por el
	usuario. 

 INSTALACION

	PASO 0
		--- DESCARGAR EL REPOSITORIO ---
		$ git clone [link del repositorio]

	PASO 1
		--- DESCARGAR LOS REQUERIMIENTOS ---
		- Primero ubicarse en la carpeta que contiene el archivo "requeriments.txt" > [C:..\PRE_ENTREGA_N3\]
		$ pip install -r requeriments.txt

	PASO 2
		--- INICIAR SERVIDOR ---
		- Primero ubicarse en la carpeta que contiene el manage.py > [C:..\PRE_ENTREGA_N3\entorno\pre_entrega_n3]
		$ python manage.py runserver
